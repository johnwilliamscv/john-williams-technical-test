# john-williams-technical-test

Test project for Boost inc featuring:
* Vue 3
* Storybook
* Pinia 
* Vite
* ESlint
* Prettier 

## Project Setup

```sh
npm install
```

### Run local database server

```sh
json-server --watch db.json --port 3000
```

### Launch project

```sh
npm run dev
```

### Launch Storybook UI

```sh
npm run storybook
```

### Run Storybook tests

```sh
npm run test-storybook
```

### Lint with [ESLint]

```sh
npm run lint
```

### Format with [Prettier]

```sh
npm run format
```
